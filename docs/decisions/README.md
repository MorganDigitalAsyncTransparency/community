# Repository-Level Decision Records

This directory contains architecture decision records that apply to the entire repository — not to a single project within it.

Project-specific ADRs live in their own directories (e.g., [discourse-observer/docs/decisions/](../../discourse-observer/docs/decisions/)).

All ADRs across the repository share a single numbering sequence to avoid conflicts. The format, status model, and conventions are defined in the [discourse-observer ADR README](../../discourse-observer/docs/decisions/README.md) and apply here as well.
